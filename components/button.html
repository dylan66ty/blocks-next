按钮 Button #

按钮是一种命令组件，可发起一个即时操作。

基本使用 #

使用 type 、plain 来定义按钮的样式

形状 #

设置属性 shape 为 circle 圆形、round 全圆角

文字按钮 #

设置属性 link 即可

无边框按钮 #

设置属性 border 为 false 即可

卡片按钮 #

设置属性 type* 为 card 即可

禁用状态 #

设置属性 disabled 为 true 即可

按钮大小 #

设置属性 size 为 mini、small、large 来定义按钮的大小
