升级指南 #

安装最新版本 #

  • 第一步，先卸载
pnpm uninstall blocks-next
  • 第二步，安装最新版本
pnpm install blocks-next@latest

无法安装最新版本 #

查看npm镜像源

npm config get registry

切换到npm官方源

npm config set registry https://registry.npmjs.org/

安装最新blocks-next

pnpm install blocks-next@latest
